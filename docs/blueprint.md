# **App Name**: Drawify

## Core Features:

- Text-to-Drawing Generation: Generates a digital drawing based on user-provided text prompts using an AI image generation model. The AI tool may add extra flourishes not present in the prompt.
- Smart Drawing Improver: Analyzes user-uploaded or hand-drawn images and uses AI to generate an improved, more polished version.
- User Authentication: Implements user authentication using Firebase Authentication (Google sign-in, guest mode).
- Drawing Storage: Saves generated and user-uploaded drawings to Firebase Storage.
- User History: Stores user history and prompts in Firebase Firestore.
- Drawing Canvas: Interactive drawing canvas for users to create and edit drawings, powered by Fabric.js.
- Drawing Saving and Export: Allows users to save generated and improved drawings, and downloads them as PNG files.

## Style Guidelines:

- Primary color: Vibrant yellow (#F9E79F) to create a fun and engaging atmosphere.
- Background color: Light yellow (#FCF4EC), a lighter tint of the primary to provide a cheerful backdrop that doesn't distract.
- Accent color: Lavender (#E6CCFF) for interactive elements and highlights, contrasting the primary with an analogous color.
- Font: 'Poppins', a geometric sans-serif font, is used for headlines and body text to give a contemporary, approachable feel.
- Use playful, cartoon-inspired icons related to drawing and creativity.
- Maintain a clean, mobile-responsive layout with rounded buttons, smooth transitions, and clear navigation.
- Incorporate fun loading animations (e.g., a sketching effect) during AI processing to keep users engaged.
